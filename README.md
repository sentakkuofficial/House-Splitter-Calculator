# Day 2 Demo - House Buying Cost Splitter

This is an interactive browser demo for Day 2 of a beginner Python course.

Students can preview the final project before building it themselves.

## Project Flow

1. Click RUN
2. Enter the house price
3. Enter the down payment percentage
4. Enter the closing cost percentage
5. Enter the number of buyers
6. See the total upfront cost and each buyer's contribution

## Example

- House price: $400000
- Down payment: 20%
- Closing costs: 3%
- Buyers: 4

The program calculates:
- down payment amount
- closing cost amount
- total upfront needed
- each buyer's share
